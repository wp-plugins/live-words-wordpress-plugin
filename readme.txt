If you simply select any word in this blog with mouse, you will see a pop over where in you can check the various dictionary list to find its synonyms.

== Description ==

* This plugin is used to find the meaning of the selected word in the blog.
* It gives us a option to get the meaning from various sites, listed in a popover.
* Once you select the site from which you want to know the meaning, the meaning is given without been re-directed from this blog page.
* After viewing the meaning, just click on the lookup tab,so that you get back to the same blog page.
    

== Installation ==

This section describes how to install the plugin and get it working.

1.Open the link http://www.spicyexpress.net/general/livewords-wordpress-plugin-now-ready-for-download-and-use/
2.Download the zip file to your local system by clicking the download link.
3.Unzip the file and copy the folder 'Meyshanlookup' into your plugin directory 
4.Now active the plugin.
5.Through admin go to option page
6.Choose the button and the location you wanted to display
7.Now you can understand that your blog is powered by Livewords and use this plugin
  Thats it!!
